
\begin{abstract}
 
This document is a complete overview of the design and implementation of the Vera C. Rubin Observatory Education and Public Outreach (EPO) program. We describe the four key audiences and specific components designed to engage with those audiences. The goal is for any person interested in the program to be able to reference this document to understand each component of the program as well as the motivation behind its design. 
\end{abstract}
